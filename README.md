# TFTP client using scapy

usage:
```
sudo python2.7 client.py [-h] [-6] -p PORT -i IPADDR
```
eg: sudo python2.7 client.py -p 31337 -i 172.17.1.125

Help:
```
python2.7 client.py -h
```

Use ```get``` and ```put``` to share  files between client and server.
